# 微观经济学II讲义

```{admonition} 课程简介
:class: tip

欢迎来到微观经济学II的学习之旅！

这本讲义采用创新的教学方法，通过日常生活中的故事和例子，特别是奶奶的养鸡场作为主要比喻，帮助您理解微观经济学的核心概念。我们从最基础的生产理论开始，逐步深入到成本分析、定价策略、市场结构等重要主题。
```

### 🎯 学习目标

本课程将帮助您掌握：- **生产理论**：理解投入产出关系和生产函数
- **成本分析**：掌握各种成本概念及其在决策中的应用
- **市场结构**：分析完全竞争、垄断等不同市场形态
- **定价策略**：学习企业如何制定最优价格
- **博弈论基础**：理解战略性决策和信息经济学

### 📖 使用指南

```{admonition} 学习建议
:class: note

- **循序渐进**：请按照章节顺序阅读，每个概念都建立在前面的基础之上
- **实践应用**：尝试将每个经济学概念应用到现实生活场景中
- **重点理解**：数学公式只是分析工具，重点在于理解背后的经济逻辑
- **互动学习**：利用每一讲中的对话形式，加深对概念的理解
```
