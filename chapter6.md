# 第六讲：奶奶，要是咱们的养鸡场"称霸"全村

```{admonition} � 本讲核心主题：垄断市场
:class: tip

本讲将探讨如果市场上只有一家养鸡场会发生什么。通过奶奶的养鸡场"称霸"全村的故事，理解垄断市场的特征、垄断厂商的决策以及垄断带来的社会成本。
```

奶奶，之前的几讲里，咱们都是假设村里有很多家养鸡场，大家竞争很激烈，谁也没法自己定价格（完全竞争）。

但是，您想象一下，要是咱们村特别偏僻，或者咱们家掌握了什么独门秘籍，导致**整个村子只有咱们一家养鸡场**，那会是什么样？

这就到了我们今天要讲的——**垄断 (Monopoly)**。

```{admonition} � 学习路径
:class: note

我们将分析垄断市场的成因，学习垄断厂商如何通过 $MR=MC$ 定价，并比较垄断与完全竞争的效率差异，最后讨论价格歧视。
```

#### 1. 为什么会变成"独苗"？——垄断的成因

为啥有的市场只有一家独大呢？通常有这么几个原因：
1.  **独占资源**：比如全村只有咱们家有一口井，没水别人养不了鸡。
2.  **政府特许**：村长只发了一张养鸡许可证，就给了咱们家。
3.  **自然垄断**：养鸡需要铺设昂贵的管道给全村送鸡蛋（假设啊），铺一套就够贵了，两家铺就浪费了，所以一家做成本最低。
4.  **技术优势**：咱们家的鸡只有咱们知道怎么养，别人学不会。

#### 2. 咱们说了算！——垄断者的定价

在完全竞争里，价格是市场定的，咱们只能接受。但在垄断市场里，**咱们是唯一的卖家，咱们说了算！**

但是，奶奶，这不代表咱们想卖多贵就卖多贵。太贵了，村民们就不买或者少买了。咱们面临的是**整个市场的需求曲线**，它是向下倾斜的。

我们要想多卖鸡，就得降价。这一降价，不光新卖的鸡便宜了，以前能卖高价的鸡现在也得跟着便宜卖（单一价格垄断）。

所以，咱们的**边际收益 (MR)** 会**低于**价格 (P)。

咱们还是用那个黄金法则：**$MR = MC$** 来决定卖多少只鸡。
算出产量后，咱们再看看需求曲线，在这个产量下，村民们愿意出多高的价格，那个价格就是咱们的定价！

**结论**：垄断厂商的定价通常**高于**边际成本 ($P > MC$)。

#### 3. 垄断好不好？——社会福利分析

奶奶，虽然垄断让咱们赚得更多了（有垄断利润），但对村里人来说，可能不是好事。
- 产量比完全竞争时少了。
- 价格比完全竞争时高了。

这就造成了一部分本来可以成交的生意没做成，经济学家叫它**无谓损失 (Deadweight Loss)**。这就是垄断带来的效率损失。

#### 4. 看人下菜碟——价格歧视

既然咱们是独家生意，咱们能不能更聪明点？
比如，对有钱的王员外，鸡蛋卖贵点；对隔壁困难的李大妈，鸡蛋卖便宜点？

这就是**价格歧视 (Price Discrimination)**。
- **一级价格歧视**：每个人都按他愿意出的最高价卖（完美榨取）。
- **二级价格歧视**：买得越多越便宜（比如批发价）。
- **三级价格歧视**：按不同的人群定不同的价（比如学生票、老人票）。

如果能做到完美的价格歧视，咱们就能把所有的消费者剩余都变成咱们的利润！

```{admonition} 📝 小结
:class: summary

- **垄断**是指市场上只有一个卖者。
- 垄断厂商遵循 **$MR = MC$** 规则，但 **$P > MC$**。
- 垄断会导致**无谓损失**，降低社会总福利。
- **价格歧视**是垄断者榨取消费者剩余的手段。
```
