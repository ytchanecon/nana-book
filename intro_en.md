# Microeconomics II Lecture Notes

```{admonition} Course Introduction
:class: tip

Welcome to the learning journey of Microeconomics II!

This lecture note adopts an innovative teaching method, using stories and examples from daily life, especially "Grandma's Chicken Farm" as the main metaphor, to help you understand the core concepts of microeconomics. We start from the most basic production theory and gradually go deeper into important topics such as cost analysis, pricing strategies, and market structure.
```
